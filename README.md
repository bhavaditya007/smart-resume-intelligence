# Smart Resume Intelligence System

Smart Resume Intelligence System is a full-stack web application designed to help users create **structured, ATS-friendly resumes** with **automated quality scoring** and **multiple professional templates**.  
The system validates user inputs, dynamically renders resumes, and provides actionable feedback to improve overall resume effectiveness.
